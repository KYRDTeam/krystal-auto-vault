# Krystal Auto Vault

Automation vault for SPL token


Commands
1. Build program
    ```
    anchor build
    ```

2. Deploy program
    ```
    anchor deploy 
    ```

3. Upgrade program
    ```
    anchor upgrade --program-id=6tgjvHkFUUUbbacEWg225H6AazxoSTso8ix9vkXFScTU target/deploy/krystal_auto_vault.so
    ```
